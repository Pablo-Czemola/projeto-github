3. Como criar uma conta no GitHub?

Para criar uma conta no GitHub, siga os seguintes passos:

  • Acesse o site do GitHub (github.com).
  • Clique no botão "Sign up" ou "Sign up for GitHub".
  • Preencha o formulário com seu nome de usuário, e-mail e senha.
  • Complete o captcha para verificar que você não é um robô.
  • Clique em "Create account" para finalizar o processo.
  • Verifique seu e-mail e siga as instruções para confirmar sua conta.
