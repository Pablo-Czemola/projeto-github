Como Criar uma Conta no GitHub e Realizar Comandos Básicos

Passos para Criar uma Conta no GitHub

  1. Acesse o site do GitHub.
  2. Clique no botão Sign up no canto superior direito.
  3. Preencha o formulário com seu nome de usuário, e-mail e senha.
  4. Clique em Sign up for GitHub.
  5. Complete o processo de verificação, se necessário, para confirmar seu e-mail.
  6. Após o login, você será direcionado para o painel do GitHub onde poderá começar a usar a plataforma.

Comandos Básicos para Começar

  1. git init - Inicialize um novo repositório Git local.
  2. git add . - Adicione todos os arquivos modificados ao índice.
  3. git commit -m "mensagem do commit" - Salve as alterações no repositório local.
  4. git remote add origin - Adicione um repositório remoto.
  5. git push -u origin master - Envie suas alterações para o repositório remoto no GitHub.
  6. git pull - Atualize seu repositório local com as mudanças do repositório remoto.
