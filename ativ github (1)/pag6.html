5. Como configurar o Git no seu computador local para se conectar ao GitHub?

Para configurar o Git no seu computador local e conectar ao GitHub, siga estes passos:

  • Instale o Git a partir do site oficial (git-scm.com).
  • Configure seu nome de usuário e e-mail no Git com os comandos:
    • git config --global user.name "Seu Nome"
git config --global user.email "seu-email@example.com"
  • Crie uma chave SSH e adicione-a à sua conta do GitHub.
  • Verifique se a configuração foi feita corretamente com:
    • ssh -T git@github.com
