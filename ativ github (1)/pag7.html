6. O que é um repositório no GitHub?

Um repositório no GitHub é um espaço onde você pode armazenar e gerenciar arquivos relacionados a um projeto. Repositórios podem conter código-fonte, documentação, e outros arquivos importantes. Cada repositório tem seu próprio histórico de commits, branches e tags, permitindo um controle detalhado sobre o desenvolvimento do projeto e facilitando a colaboração entre desenvolvedores.
