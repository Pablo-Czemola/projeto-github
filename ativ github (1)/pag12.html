11. Quais são os principais comandos do Git que são usados com frequência?

Alguns dos principais comandos do Git incluem:

  • git init - Inicializa um novo repositório Git.
  • git clone - Clona um repositório remoto para sua máquina local.
  • git add - Adiciona arquivos ao índice para o próximo commit.
  • git commit - Cria um novo commit com as alterações adicionadas.
  • git push - Envia commits locais para o repositório remoto.
  • git pull - Atualiza seu repositório local com as últimas alterações do remoto.
