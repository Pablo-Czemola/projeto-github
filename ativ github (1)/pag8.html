7. Como criar um novo repositório no GitHub?

Para criar um novo repositório no GitHub, siga estes passos:

  • Faça login na sua conta do GitHub.
  • Clique no botão "New" na página de repositórios.
  • Preencha o formulário com o nome do repositório, uma descrição (opcional) e selecione se deseja que o repositório seja público ou privado.
  • Adicione um README, .gitignore ou uma licença se desejar.
  • Clique em "Create repository" para finalizar a criação.
