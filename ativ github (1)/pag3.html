Diferenças entre Git e GitHub

2. Quais são as diferenças entre Git e GitHub?

Git e GitHub são frequentemente confundidos, mas desempenham papéis diferentes no desenvolvimento de software. Git é um sistema de controle de versão distribuído, que permite a você acompanhar as alterações no código-fonte ao longo do tempo e colaborar com outros desenvolvedores localmente. GitHub, por outro lado, é uma plataforma de hospedagem de código-fonte que usa Git como seu sistema de controle de versão. GitHub oferece um ambiente online para armazenar repositórios Git, colaborar com outros desenvolvedores, e gerenciar projetos de software com recursos adicionais como pull requests, issues e wikis.
