17. Como revisar e aceitar um pull request?

Para revisar e aceitar um pull request no GitHub:

  • Vá para a seção de "Pull requests" do repositório.
  • Selecione o pull request que deseja revisar.
  • Leia as alterações e comentários.
  • Clique em "Files changed" para ver as mudanças detalhadas.
  • Clique em "Merge pull request" para aceitar as mudanças.
  • Adicione uma mensagem de merge, se desejar, e clique em "Confirm merge".
