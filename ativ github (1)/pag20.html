19. Como usar o sistema de projetos do GitHub para gerenciar tarefas?

O sistema de projetos do GitHub permite organizar e gerenciar tarefas usando quadros, cartões e colunas. Para usar o sistema de projetos:

  • Vá para a seção "Projects" do repositório.
  • Clique em "New project" para criar um novo projeto.
  • Adicione colunas ao projeto para organizar tarefas, como "To do", "In progress" e "Done".
  • Clique em "Add cards" para adicionar issues e pull requests aos cartões.
  • Arraste e solte os cartões para atualizar o progresso.
