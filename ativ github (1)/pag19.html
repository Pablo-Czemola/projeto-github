18. O que são issues e como abrir uma issue no GitHub?

Issues são uma maneira de registrar e rastrear bugs, melhorias ou outras tarefas relacionadas a um projeto no GitHub. Para abrir uma issue:

  • Vá para a página do repositório no GitHub.
  • Clique em "Issues".
  • Clique em "New issue".
  • Preencha o título e a descrição da issue.
  • Clique em "Submit new issue" para criar a issue.
