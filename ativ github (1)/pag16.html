15. Como fazer um fork de um repositório?

Para fazer um fork de um repositório no GitHub:

  • Acesse o repositório que deseja fork.
  • Clique no botão "Fork" no canto superior direito da página.
  • Escolha a conta onde deseja criar o fork.
  • O GitHub criará uma cópia do repositório em sua conta.
