9. O que significa clonar um repositório e como isso é feito?

Clonar um repositório significa criar uma cópia local do repositório remoto para o seu computador. Isso permite que você trabalhe no projeto localmente, faça alterações e sincronize essas alterações com o repositório remoto. Para clonar um repositório, use o comando:

git clone https://github.com/usuario/repositorio.git
