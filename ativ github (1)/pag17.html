16. O que é um pull request e como criar um pull request?

Um pull request é uma solicitação para integrar alterações de uma branch para outra. Para criar um pull request:

  • Faça um commit e push das alterações para uma branch no GitHub.
  • Vá para a página do repositório no GitHub.
  • Clique em "Pull requests" e depois em "New pull request".
  • Escolha a branch de origem e a branch de destino.
  • Descreva as alterações e clique em "Create pull request".
