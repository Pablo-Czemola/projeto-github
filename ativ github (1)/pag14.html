13. O que são branches e como criar uma nova branch no GitHub?

Branches (ou ramificações) são uma maneira de divergir do código principal e trabalhar em funcionalidades ou correções sem afetar o código principal. Para criar uma nova branch no GitHub, siga estes passos:

  • Acesse o repositório no GitHub.
  • Clique no menu suspenso de branches e insira um nome para a nova branch.
  • Clique em "Create branch" para finalizar.
