12. Como fazer commits e qual a importância de uma mensagem de commit clara e descritiva?

Para fazer um commit, use o comando:

git commit -m "Sua mensagem de commit"

É crucial que suas mensagens de commit sejam claras e descritivas. Isso ajuda você e outros colaboradores a entenderem o histórico de alterações e o propósito de cada mudança. Uma boa mensagem de commit deve explicar o "porquê" da alteração, não apenas o "o quê" foi alterado.
