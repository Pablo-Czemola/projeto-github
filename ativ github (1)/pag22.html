Principais Comandos do GitHub

GitHub é uma plataforma de hospedagem para controle de versão usando Git. Além dos comandos do Git, GitHub oferece ferramentas adicionais para colaborar em projetos, gerenciar pull requests, e muito mais. Aqui estão alguns comandos comuns utilizados no GitHub:

  • git fork - Cria uma cópia do repositório em sua conta GitHub.
  • git pull request - Cria um pedido de inclusão de alterações no repositório original.
  • git merge - Integra alterações de uma branch para outra.
  • git rebase - Reaplica commits de uma branch sobre outra.
  • git tag - Marca pontos específicos na história do repositório.
